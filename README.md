Descripción del proyecto:
Este proyecto presenta una aplicación web diseñada para predecir el estado de salud de una persona, clasificándola como sana o en riesgo. Para ello, se entrenaron tres modelos de Machine Learning: K-Nearest Neighbors (KNN), Multi-Layer Perceptron (MLP) y Regresión Lineal.

El entrenamiento se realizó utilizando un dataset obtenido de Kaggle que incluye variables relevantes para el bienestar físico, tales como: la edad, el género, índice de masa corporal (BMI), pasos diarios, horas de sueño, agua consumida, calorías, si es fumador, si consume alcohol, frecuencia cardíaca en reposo, presión arterial sistólica, presión arterial diastólica y colesterol. Todas estas variables se le piden al usuario para que los modelos predigan si su estado de salud es bueno o malo (En riesgo o sano).

La aplicación solicita estos datos al usuario a través del frontend y los envía al backend, donde los modelos de predicción procesan la información y devuelven un diagnóstico sobre su estado de salud. Cabe aclarar que la precisión de las predicciones van ligadas al dataset y su naturaleza.

Pasos para el despliegue de la aplicación:

 1. Requisitos para ejecutar el proyecto: Para ejecutar correctamente la aplicación es necesario contar con Python  3.10 o una versión superior, ya que el backend está desarrollado con Flask y requiere este entorno. También se debe tener instalado pip, el cual es el gestor de paquetes de Python, para permitir la instalación de todas las dependencias, también es necesario disponer de Git para clonar el repositorio desde GitHub y poder correr correctamente el proyecto, así como un navegador web actualizado, para acceder e interactuar con el frontend.
 2. Clonar el repositorio: Escriba en en la terminal: git clone, luego en la opcion "<Code>" del repositorio seleccione el endpoint de la opcion HTTPS y peguelo para clonar el repositorio actualizado.
 3. Instalar las dependencias: En la terminal escriba: pip install -r requirements.txt, estas dependencias incluyen Flask y las librerías de Machine Learning usadas para cargar y ejecutar los modelos.
 4. Inicializar el backend: Ubicarse en el archivo app_salud.py, en este archivo se visualizará la API de flask donde se encuentran los modelos de machine learning a utilizar. 
 Para inicializarlo escriba en la terminal: flask --app app_salud run, después de esto aparecerá un endpoint el cual indica que la página ya está inicializada correctamente. 
 5. Acceder al frontend: El frontend está compuesto por 2 archivos que son api.js e index.html. En el primero de ellos se encuentran todas las variables que se pedirán en el sitio web para poder generar la predicción, las cuales se convierten en números cuando es necesario, como por ejemplo las variables de fumador, genero o antecedentes familiares, posterior a eso las envía al servidor mediante un POST, muestra un loader mientras procesa los datos y, al recibir la respuesta, muestra los resultados en la  pantalla. Además, permite activar o desactivar el modo oscuro cambiando la apariencia del sitio. El segundo de ellos construye una interfaz visual donde el usuario ingresa las variables que se le piden, aplica estilos con CSS, muestra un loader mientras se procesan los datos y presenta los resultados enviados por el backend. Además, incluye un botón para cambiar entre modo claro y oscuro, integra Bootstrap, Google Fonts y FontAwesome, y carga el archivo api.js que envía las variables al servidor para obtener las predicciones.
 6. Probar el sitio web: Para dirigirse al sitio web acceda a la carpeta creada del proyecto, alli se debe abrir el archivo llamado "index.html". Ya abierto el sitio ingrese los datos y prediga su estado de salud.
